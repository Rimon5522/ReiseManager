# -ReiseManager-
------------------------------------------------------------
# 1.Beginn der Planung
- Da ich noch nicht so erfahren bin, habe ich mir überlegt, eine Mischung aus allem zu machen, was wir bisher gelernt haben. So kann man zeigen, dass man alles verstanden hat oder selbst besser erkennen, was man noch nicht beherrscht.
- An sich ist das eine gute Idee, weil man sich dadurch richtig mit den ganzen Themen auseinandersetzen und üben kann.
- Also kam ich auf die Idee, ein Programm zu schreiben, bei dem man Reisetickets kaufen kann.

-------------------------------------------------------------
# 2.Was soll alles im Programm enthalten sein (Grundgerüst)
- Gerne würde ich im Programm ein Login zum Anmelden haben. Dann eine Personenabfrage. Zudem soll anhand von drei festgelegten Reisezielen gefragt werden, welches man auswählen möchte.
- Danach soll gefragt werden, wie viele Personen mitreisen und wie alt diese Personen sind. Ich möchte außerdem einen Rabatt für Personen unter 18 Jahren einbauen.
- Im Anschluss soll anhand der Anzahl der Personen eine Rechnung erstellt werden.
- Zusätzlich möchte ich hinzufügen, ob man Handgepäck oder einen großen Koffer dabei hat, damit dieser zur Rechnung addiert wird.
- Ein Wetterbericht für den Zeitraum der Reise soll ebenfalls angezeigt werden. Außerdem eventuell eine Auswahl oder Zuweisung der Sitzplätze im Flugzeug.
